NetworkLagMonitor
=================

Code and schematics for an Arduino network lag monitor that uses three LEDs to display ping times

Components Needed
-----------------
- Ethernet Sheild
- 3 220ohm resistors
- 3 jumper wires
- 1 green, yellow, and red LED
 
Libaries Needed
-----------------
[Arduino Ping](https://github.com/BlakeFoster/Arduino-Ping) - I used v 2.1
